# 澳门住户资讯科技使用状况分析

## 项目概述

这是一个基于2024年澳门统计数据的交互式数据可视化应用，采用学术风格设计，专注于展示澳门居民在资讯科技使用方面的统计特征和模式分析。

## 数据来源

- 数据集：SC_UTI_FR_2024_Y.xls
- 来源：澳门统计暨普查局
- 涵盖内容：10个主要统计维度，包括年龄、教育程度、活动状态、职业等人口统计学特征

## 核心可视化组件

### 章节式叙事结构

#### 第一章：数字鸿沟的代际差异
- **核心问题**: 不同年龄段的科技使用差异有多大？
- **交互筛选**: 年龄段按钮（18-24岁、25-44岁、45岁以上、全年龄段）
- **可视化**: 雷达图对比分析 + 动态洞察解读

#### 第二章：科技产品的使用偏好
- **核心问题**: 哪些科技产品在澳门最受欢迎？
- **交互筛选**: 产品类型按钮（移动设备、电脑应用、互联网服务、网上购物）
- **可视化**: 桑基图流向分析 + 关键词云图

#### 第三章：多视角综合分析
- **核心问题**: 从不同视角审视澳门的数字化进程
- **交互筛选**: 分析视角按钮（人口统计、经济活动、教育水平、综合概览）
- **可视化**: 树状图层次展示

#### 第四章：趋势预测与展望
- **核心问题**: 这些趋势将如何演变？
- **交互筛选**: 时间跨度按钮（短期1-2年、中期3-5年、长期5-10年、当前基准）
- **可视化**: 时间序列趋势预测图 + 预测解读

#### 第五章：政策建议与行动计划
- **核心问题**: 如何缩小数字鸿沟，提升澳门整体数字化水平？
- **交互筛选**: 政策领域按钮（教育培训、基础设施、产业支持、综合策略）
- **可视化**: 政策优先级评估图 + 实施路线图

### 基础概览组件

#### 1. 统计维度关系网络
- **图表类型**: 网络图（Network Graph）
- **分析目的**: 展示不同统计维度间的内在关联性和数据结构层次
- **设计特点**: 采用力导向布局算法，节点大小和颜色编码维度重要性

#### 2. 数据收集层次结构
- **图表类型**: 桑基图（Sankey Diagram）
- **分析目的**: 呈现统计数据的分类体系和信息流向
- **设计特点**: 清晰展示从人口统计到具体应用的数据层次关系

#### 3. 统计模式相关性矩阵
- **图表类型**: 热力图（Heatmap）
- **分析目的**: 分析不同统计维度间的相关强度
- **设计特点**: 使用连续色阶编码相关系数，数值标注提高精确性

#### 4. 关键词分析
- **图表类型**: 文字云（Word Cloud）
- **分析目的**: 数据集中高频出现的统计概念可视化
- **设计特点**: 字体大小反映词频，颜色渐变增强视觉层次

## 设计原则

### 学术风格
- 采用Source Sans Pro字体系列
- 配色方案：蓝色系主色调，灰色系辅助色
- 布局简洁，信息密度适中

### Data-Ink Ratio优化
- 移除图表冗余装饰元素
- 最小化非数据墨水使用
- 保持视觉元素的功能性

### 响应式布局
- 左右列布局设计
- 一页完整展示所有内容
- 移动端自适应

## 核心发现

1. **数据覆盖全面**: 涵盖10个主要统计维度，展现澳门居民科技使用全景
2. **人口统计维度突出**: 年龄、教育程度和活动状态是三大核心分析维度
3. **职业影响显著**: 互联网和通信工具使用情况受到职业特征显著影响
4. **经济发展映射**: 商业用途科技应用反映澳门经济发展特征

## 数据解读

这份统计数据反映了澳门作为现代化城市的科技使用特征：

- **人口结构影响**: 年龄和教育程度显著影响科技 adoption 率
- **经济活动关联**: 就业人口的科技使用与职业类型密切相关
- **服务应用广泛**: 从基本通信到在线购物，科技服务渗透各个生活层面
- **技能差异明显**: 电脑技能水平影响科技使用的深度和广度

## 学术价值

该可视化分析为理解澳门数字化转型提供了数据支撑，有助于：
- 政策制定者制定针对性的科技普及策略
- 教育机构优化相关课程设置
- 企业了解市场需求特征
- 研究者进行更深入的比较分析
