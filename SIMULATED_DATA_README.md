# 澳门ICT使用情况模拟数据分析

## 数据概述

`simulated_samples.xlsx` 包含了1000个模拟的澳门居民ICT使用情况样本，这些数据是基于原数据集的比例关系通过概率抽样生成的。

### 数据结构

**总样本量**: 1,000
**变量数量**: 21个

#### 人口统计变量
- `age_group`: 年龄组 (3-14, 15-24, 25-34, 35-44, 45-54, 55-64, 65-74, >=75)
- `gender`: 性别 (male/female)
- `education_level`: 教育水平
- `economic_status`: 经济状况
- `occupation`: 职业

#### ICT使用变量 (二元变量: 0/1)
- `mobile_phone`: 移动电话使用
- `computer`: 电脑使用
- `tablet`: 平板电脑使用
- `smart_tv`: 智能电视使用
- `no_device`: 无互联网设备
- `communication_apps`: 通讯/社交应用
- `social_media`: 社交媒体
- `online_shopping`: 在线购物
- `online_banking`: 网上银行/电子支付
- `cloud_services`: 云服务
- `iot_devices`: 物联网设备
- `streaming_services`: 流媒体服务
- `video_conferencing`: 视频会议
- `remote_work`: 远程工作

## 可视化功能

### 1. 密集数据点点阵
- **用途**: 探索多变量关系，支持条件筛选
- **特点**:
  - 使用移动电话和电脑使用作为坐标轴
  - 不同年龄组用不同颜色编码
  - 性别通过形状区分
  - 支持年龄组、性别、互联网接入类型的筛选
  - 悬浮显示详细信息

### 2. 箱线图与点图结合
- **用途**: 分析变量分布特征
- **特点**:
  - 同时显示按年龄组和性别的分布
  - 箱线图展示统计分布
  - 散点显示个体数据点
  - 支持变量选择和条件筛选

### 3. 相关性热力图
- **用途**: 探索ICT使用模式之间的关系
- **特点**:
  - 显示所有数值变量之间的相关系数
  - 颜色编码相关性强度
  - 支持条件筛选

## 数据质量评估

✅ **优点**:
- 样本量充足 (n=1000)
- 无缺失值
- 变量类型合适 (分类变量 + 二元数值变量)
- 分布合理，符合澳门人口特征

✅ **适合的可视化类型**:
- 密集散点图 (条件筛选，高亮区域)
- 箱线图 + 点图 (分布分析)
- 热力图 (相关性分析)
- 交互式仪表板 (多维度探索)

## 使用方法

### 运行完整分析应用（推荐）

主应用现在包含了模拟数据可视化功能：

```bash
python macau_tech_analysis.py
```

然后在浏览器中访问 `http://127.0.0.1:8050/`

在第6章"Simulated Data Interactive Analysis"中可以找到所有模拟数据可视化功能。

### 单独运行模拟数据可视化

如果只想查看模拟数据可视化：

```bash
python viz_simulated_data.py
```

然后在浏览器中访问 `http://127.0.0.1:8051/`

### 数据分析

```bash
python analyze_simulated_data.py
```
